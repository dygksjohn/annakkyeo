# 평가 리포트 — 안낚여(annakkyeo)

한국어 스미싱 **판정 + 근거 설명** 소형 LLM 파인튜닝의 평가 리포트 (초안, 2026-07-08).
원천 기록: [평가_실험로그.md](평가_실험로그.md) (EXP-001·004·006~012). 수치는 집계 지표만 기재(문자 원문·개인정보 없음).

---

## 1. 개요

판정만 하는 스팸 필터와 달리, 문자를 `{verdict, risk_factors, explanation}` 로 출력해 **"왜 위험한지"까지 생성**하는 것이 목표다. 성공 기준은 **파인튜닝 모델 F1이 GPT-4o-mini 베이스라인(엄격 F1 0.881)에 근접·초과**하는 것.

**결과: 파인튜닝 6종 전부 베이스라인을 초과했고(F1 0.974~0.993), 설명 품질(사실성 4.48/5)까지 정량 입증했다.**

## 2. 실험 설정

- **테스트셋**: `eval_sample.csv` 600건 (Kor-Smishing 균형 샘플, 스미싱 300/정상 300, seed=42). GPT 베이스라인과 **동일 셋** → apples-to-apples. 학습셋에서 제외(누수 없음).
- **지표**: 이진 분류. 양성=스미싱. **엄격(strict)** 매핑 = `smishing`만 양성. Precision(오탐 억제)·Recall(미탐 억제)·F1.
- **추론**: 4bit(nf4) QLoRA, greedy, max_new_tokens=256, `enable_thinking=False`. RTX 3050 6GB / transformers 4.56.2.
- **모델**: 주력 Qwen3-1.7B, 비교 Kanana-1.5-2.1B, 스케일업 Qwen3-4B-Instruct-2507. 각 v1(스미싱 편중 데이터)·v2(하드네거티브 1:1 균형) 어댑터.

## 4. 핵심 발견

**① 파인튜닝이 base 모델 격차를 압도한다.**
zero-shot에선 Qwen3-1.7B(0.722) ≫ Kanana-2.1B(0.545)로 벌어졌으나, 동일 SFT 데이터로 파인튜닝하면 둘 다 0.97~0.99로 수렴했다. **base 선택보다 SFT 데이터 품질이 지배적**. 파싱 실패(형식 미준수)도 4.5~33.7% → 0%로 완전 해소.

**② 소형 모델로 API급 성능.** 주력 1.7B QLoRA(0.974)가 GPT-4o-mini(0.881)를 **+0.093 초과**. 파라미터 1000분의 몇 수준의 로컬 모델이 상용 API를 넘었다.

**③ 정밀도(오탐↓) 개선은 두 축.** 오탐(FP)은 zero-shot의 유일한 약점(Qwen3 FP=199)이었는데, ⓐ **v2 하드네거티브 재균형**(전 모델 FP↓)과 ⓑ **스케일업**(1.7B→4B, FP 15→8)이 각각 기여. 둘을 겹치면(**Qwen3-4B v2**) **FP=0**.

**④ 최고점 2개, 운영점이 다르다.**
- **Kanana-2.1B v2** (F1 0.993): 재현율 우선(R 0.997, 미탐 1건). 스미싱을 놓치면 안 되는 환경.
- **Qwen3-4B v2** (F1 0.993): **정밀도 완벽(1.0, 오탐 0)**. GPT의 무오탐 특성을 재현율 0.987(GPT는 0.787)로 달성. 오경보 0이 필요한 환경.

**⑤ 차별점(설명 품질)을 정량 입증.** 주력 v1 설명을 gpt-4o로 채점: **사실성 4.48** / 구체성 2.98 / 실행가능성 3.68 (종합 3.71/5, 553건). 판정뿐 아니라 **근거를 사실적으로 설명**함을 수치로 보였다. 약점은 구체성 → v2 설명 라벨 강화 여지.

## 5. 한계 (정직한 일반화 검증)

위 수치는 **학습과 동일 출처 분포**(Kor-Smishing) held-out이다. 별도로 수집한 **2024 실 스미싱 24건(다른 분포)** 에서는 재현율이 **0.58~0.67로 급락**했다(신유형 지인·기관 사칭 미탐 집중, 택배류는 잘 탐지). 즉 in-distribution F1 0.99는 강하나 **신유형 일반화 갭이 존재**한다. N=24 소표본·정상 held-out 미확보라 확정적이진 않다.
→ **v3에서 최신·지인사칭 스미싱 다양화** 권장. 리포트에서 0.99를 절대 성능으로 과장하지 말 것.

## 6. 결론·권고

- **성공 기준 달성**: 소형 파인튜닝 모델이 GPT 베이스라인을 초과(+판정+설명).
- **배포 권고**: 1.7B로도 GPT 초과 → 속도·메모리상 **Qwen3-1.7B를 주력**으로. 데모 기본값은 용도로 선택 — 미탐 최소=재현율형(v1), 오경보 최소=정밀도형(v2/4B v2).
- **다음(v3)**: 신유형·정상 하드네거티브 확대로 **일반화 갭** 축소. 별도 실 held-out(정상 포함) 확보 후 재평가.
