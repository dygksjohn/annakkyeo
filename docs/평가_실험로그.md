# 평가 실험 로그 — 안낚여 (annakkyeo)

> 계획서 산출물 #2(평가 리포트)의 원천 기록. **지표(숫자)만 기록**하며 문자 원문·개인정보는 남기지 않는다.
> 평가 코드: `src/baseline_gpt.py`, `src/metrics.py`. 재현: `python -m src.baseline_gpt --n-per-class 100`

---

## 이진 판정(스미싱 탐지) 지표 정의

- 양성(positive) = 스미싱(class 1), 음성 = 정상 ham(class 0)
- 모델 출력 verdict(smishing/suspicious/normal)를 이진으로 매핑해 평가:
  - **엄격(strict)**: `smishing` 만 양성
  - **관대(lenient)**: `suspicious` 도 양성(경고)으로 취급
- Precision = 스미싱이라 판정한 것 중 실제 스미싱 비율 (오탐 억제)
- Recall = 실제 스미싱 중 잡아낸 비율 (미탐 억제)

| 항목 | 값 |
|------|-----|
| 일자 | 2026-07-05 |
| 모델 | gpt-4o-mini (temperature=0, JSON 모드) |
| 평가셋 | Kor-Smishing 균형 샘플 600건 (스미싱 300 / 정상 300), seed=42 |
| 프롬프트 | 공용 템플릿 (docs/스키마_프롬프트_설계.md §2) |
| 파싱 실패 | 0 / 600 |

**결과**

| 매핑 | Accuracy | Precision | Recall | F1 | Macro-F1 | 혼동행렬 (TP/FN/FP/TN) |
|------|----------|-----------|--------|-----|----------|------------------------|
| 엄격 (smishing만) | 0.8933 | **1.0000** | 0.7867 | 0.8806 | 0.8921 | 236 / 64 / 0 / 300 |
| 관대 (suspicious 포함) | 0.9517 | 0.9274 | **0.9800** | 0.9530 | 0.9516 | 294 / 6 / 23 / 277 |

> 참고: 200건 예비 실행에서도 엄격 F1=0.8827 로 거의 동일 → 수치 안정적. 위 600건이 확정값.

**해석**
- 엄격 매핑에서 **오탐(FP)=0** — GPT는 정상 문자를 스미싱으로 잘못 판정하지 않음. 대신 미탐(FN)=64.
- 미탐 64건 중 대부분은 GPT가 `suspicious`로 분류한 것으로 보임 → 관대 매핑에서 recall이 0.79→0.98로 급등(FP는 0→23으로 증가).
- 이는 판정 임계(스미싱 vs 의심)의 트레이드오프를 보여줌. 파인튜닝 모델은 이 경계를 데이터로 학습해 개선하는 것이 목표.

**한계 / 다음 단계**
- 이 평가셋은 **Kor-Smishing(공개 실데이터) 샘플**로, 계획상 최종 test셋(별도 held-out 실데이터/KISA)과 구분됨. 예비 베이스라인 성격.
- 아직 측정 안 함: 베이스 sLLM zero-shot / few-shot (GPU 필요, 7/8 이후), 설명 품질 LLM-judge.
- 이 F1(엄격 0.881)이 1~3B 파인튜닝 모델이 근접·초과해야 할 기준선.

### 설명 품질 LLM-judge (A2, 이 프로젝트 핵심 차별점)

판정(F1) 외에 모델이 생성한 `explanation`의 품질을 GPT judge(gpt-4o)로 3축 1~5점 채점.
입력: `outputs/qwen3-1.7b-qlora-v1/predictions.jsonl`, 코드: `src/judge_explanation.py`.

| 축 | 평균(1~5) | 해석 |
|----|-----------|------|
| 사실성 factuality | **4.481** | 설명이 문자 내용과 사실 일치 — 환각 거의 없음(강점) |
| 구체성 specificity | 2.976 | 다소 일반적 — 개선 여지(약점) |
| 실행가능성 actionability | 3.684 | 대응행동 안내 준수 |
| **종합** | **3.714** | 1.7B 소형 모델 치고 양호 |

- 채점 대상 **553 / 600** (judge 응답 오류 47건 제외; predictions 파싱실패·빈설명은 0).
- 의의: **판정 F1 0.974 + 설명 사실성 4.48/5** — "판정만 하는 스팸 필터"와 달리 "왜 위험한지"를 사실적으로 설명함을 정량 입증. 약점(구체성 2.98)은 v2에서 설명 라벨의 구체성 강화로 개선 여지.
